# Python Calculator

A simple calculator application in Python with automated testing and continuous integration (CI) using GitHub Actions.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with divide-by-zero protection)

## Requirements
- Python 3.10+ (tested on 3.10, 3.11, 3.12)
- `pip`
